### useRef hook
- useRef() hook is used so that we can prevent a variable to not re render when our whole react site re renders. Hence the value inside the variable doesn't resets.

- ref = {ref_value}, we can now use the ref_value to access the given element without the need for DOM manipulation.

- we can make very interactive websites using useRef hook, refer to react documentation for examples and boiler plate code for the same.
